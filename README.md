**To-Do List Application** built in Python with a **interactive pie chart** to visualize task progress!  
---

### Features
-  Add new tasks  
-  View all tasks with status.
-  Mark tasks as completed  
-  **Pie Chart** showing completion percentage (using Matplotlib)  
-  Clean & colorful terminal interface with emojis  
---

###  Tech Stack
- Python 3
- Matplotlib (for visualization)

---

###  How to Run
1.Download the python file and open it in any code editor.  
2.Install matplotlib by typing "pip install matplotlib" in the terminal.  
3.Run the python file.
